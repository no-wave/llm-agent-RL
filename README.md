## 책 소개

LLM을 처음 진지하게 다뤄본 사람들이 공통적으로 경험하는 순간이 있다. ChatGPT가 복잡한 질문에 막힘없이 답하는 모습을 보거나, Claude가 긴 문서를 단숨에 분석하고 코드를 작성하는 장면을 보며 "이 모델이 스스로 목표를 세우고 행동할 수 있을까"라는 질문을 품는 순간이다. LLM은 분명 놀랍다. 방대한 지식을 언어로 유창하게 표현하고, 맥락을 이해하며, 복잡한 추론을 수행하는 능력은 불과 몇 년 전만 해도 상상하기 어려웠다.

문제는 그 다음이다. LLM은 질문에 답하는 데는 뛰어나지만, 스스로 목표를 설정하고, 환경과 상호작용하며, 피드백을 통해 전략을 개선하는 데는 본질적인 한계가 있다. 막상 에이전트를 구현하려 하면 순차적 의사결정, 환경 상태의 표현, 보상 설계, 탐색과 활용의 균형 같은 문제들이 쏟아진다. 이론은 추상적이고, LLM과 강화학습을 잇는 다리는 생각보다 복잡하다. 많은 실무자들이 바로 그 지점에서 멈춘다. 이 책은 그 지점을 통과하기 위해 썼다.

이 책은 LLM과 강화학습이 결합하여 자율적으로 행동하는 에이전트를 만드는 원리와 방법을 이론과 실전 모두에서 체계적으로 다룬다. 단순히 방법론을 나열하는 것을 넘어, 각각의 접근법이 왜 등장했고, 어떤 문제를 해결하며, LLM 에이전트의 어떤 역량과 연결되는지를 함께 짚는다.

책은 다섯 개의 파트로 구성된다.

Part 1: Agent RL 들어가기 전에는 이 책 전체의 출발점이다. LLM Agent RL을 본격적으로 탐구하기 전에 반드시 짚어야 할 배경 개념과 전체적인 흐름을 정리한다. LLM 에이전트와 강화학습의 접점이 이미 익숙하다면 빠르게 훑고 넘어가도 좋고, 처음 접하는 독자라면 충분히 시간을 들여 읽기를 권한다. 이후의 모든 내용이 이 위에 세워진다.

Part 2: LLM RL에서 에이전트 RL로의 전환에서는 텍스트 생성 품질을 높이는 데 집중했던 기존 LLM RL이 어떻게 환경과 상호작용하는 에이전트 RL로 확장되는지를 다룬다. 마르코프 의사결정 과정, 환경 상태, 행동 공간, 전이 동역학, 보상 함수, 학습 목표, RL 알고리즘까지, 에이전트 RL의 근간을 이루는 핵심 개념들을 체계적으로 쌓아 올린다.

Part 3: 에이전트 RL: 역량 관점에서는 자율적 에이전트가 갖춰야 할 역량들을 하나씩 탐구한다. 계획, 도구 사용, 메모리, 자기 개선, 추론, 인식까지, 강화학습이 LLM 에이전트의 각 역량을 어떻게 강화하는지를 역량별로 깊이 있게 살펴본다.

Part 4: 에이전트 RL: Search Agent에서는 역량 관점을 넘어 실제 과업 중심으로 시야를 전환한다. 검색과 연구라는 대표적인 에이전트 과업을 중심으로, 오픈소스 RL 방법론부터 폐쇄형 모델의 RL 전략까지 실전에서 작동하는 Search Agent의 전모를 살펴본다.

Part 5: 에이전트 RL: Code Agent에서는 강화학습이 만들어낸 가장 실용적이고 강력한 에이전트 형태를 다룬다. 코드 생성을 위한 RL부터 반복적 코드 정제, 소프트웨어 자동화를 위한 RL까지, Code Agent가 단순한 코드 완성 도구를 넘어 자율적인 소프트웨어 엔지니어로 진화하는 과정을 단계적으로 탐구한다.

이 책은 총 3편으로 기획된 AI Master: 강화학습 시리즈의 세 번째 편이다. 1편이 클래식 강화학습의 기초를 다졌고, 2편이 딥러닝과 결합한 현대적 알고리즘들을 체계적으로 쌓아 올렸다면, 이 책은 그 위에서 LLM과 강화학습이 만나는 최전선을 탐구한다.

지금 우리는 에이전트의 시대 한가운데 서 있다. GPT, Claude, Gemini 같은 대규모 언어 모델이 단순한 질의응답 도구를 넘어, 도구를 사용하고, 계획을 세우며, 복잡한 작업을 자율적으로 수행하는 에이전트 시스템의 핵심이 되었다. DeepSeek-R1이 강화학습만으로 추론 능력을 극적으로 끌어올리고, OpenAI o3가 복잡한 소프트웨어 엔지니어링 과업을 자율적으로 수행하는 지금, LLM과 강화학습의 결합은 선택이 아닌 필수가 되었다.
그러나 LLM 단독으로는 장기적 목표를 위한 순차적 의사결정과 환경과의 상호작용을 통한 피드백 기반 학습이 불가능하다. 강화학습은 바로 그 빈자리를 채운다. RLHF가 LLM을 인간의 가치에 정렬하고, GRPO와 같은 방법론이 추론 능력을 비약적으로 높이며, 강화학습 기반 에이전트가 복잡한 환경에서 장기 목표를 달성하는 시대가 바로 지금이다.

에이전트의 이름을 아는 것과, 그 작동 원리를 손으로 직접 이해하고 설계해본 것은 전혀 다른 역량이다. 이 책을 마칠 때쯤이면, 새로운 에이전트 논문을 읽고 그 설계 원리를 꿰뚫어 볼 수 있는 눈과, 직접 에이전트를 구성할 수 있는 토대가 만들어져 있을 것이다.
